Projects

  • ARify: Leveraging Narrated Instructional Videos to Create Augmented Reality Tutorials for Procedural Tasks

    Semi-automatic AR authoring system that converts narrated instructional videos into spatially anchored AR tutorials.

    • Integrated object detection, segmentation, pose estimation, hand tracking, and VLM-based task parsing into an AR tutorial generation workflow.
    • Built a desktop authoring workflow for refining tutorial structures and spatial placement of AR guidance elements.
  • GesPrompt: Leveraging Co-Speech Gestures to Augment LLM-Based Interaction in Mixed Reality

    Multimodal XR interaction workflow that combines speech, gesture recognition, and LLM-based prompt tuning.

    • Designed a co-speech gesture framework that extracts spatial parameters from gestures to clarify ambiguous speech commands.
    • Improved XR interaction accuracy by jointly processing gesture and speech inputs through an LLM agent.
  • agentAR: Creating Augmented Reality Applications with Tool-Augmented LLM-based Autonomous Agents

    Tool-augmented LLM agent that supports natural-language AR application creation.

    • Built a machine-learning tool library for agent-based AR workflows by synthesizing common structures from AR research systems.
    • Tested robustness by recreating six existing AR applications and creating six new AR applications with different ML algorithm combinations.
  • Ubi Edge: Authoring Edge-Based Opportunistic Tangible User Interfaces in Augmented Reality

    AR system for detecting 3D edges in physical environments and turning them into tangible user interface inputs.

    • Created 3D edge point clouds using RGBD-based edge detection, YOLO segmentation, BundleTrack 6D pose tracking, and edge-based ICP.
  • ARnnotate: An Augmented Reality Interface for Collecting Custom Datasets of 3D Hand-Object Interaction Pose Estimation

    AR-based workflow for collecting custom hand-object pose estimation datasets.

    • Implemented CenterPose object pose estimation and OpenPose hand detection networks, paired with Unity3D interaction on Oculus Quest 2.
  • Scalar: Authoring Semantically Adaptive Augmented Reality Experiences in Virtual Reality

    AR/VR workflow for defining and validating semantically adaptive AR experiences in synthetic VR environments.

    • Integrated 3D semantic understanding, YOLO detection, and an SVM-based adaptation model for AR deployment.
  • Gesturar: An Authoring System for Creating Freehand Interactive Augmented Reality Applications

    Workflow for authoring freehand interactive AR experiences through gesture demonstration and visual programming.

    • Implemented CNN-based gesture detection and Siamese-network gesture comparison with PyTorch and Unity Barracuda.
